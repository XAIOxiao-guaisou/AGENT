## 🔍 核心概念

### 什么是 PLAN.md?

`PLAN.md` 是项目的**需求文档**和**开发蓝图**,包含:

1. **项目基础信息** - 名称、目标、描述
2. **架构蓝图** - 文件结构、模块划分
3. **核心逻辑要求** - 功能需求、业务规则
4. **技术栈** - 使用的库和工具
5. **完成标准** - 验收条件
